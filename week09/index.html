    Reading 1: Chapter 8 - Transitions

    • Preface: Transforms are fun and a handy way to move things around.
    • Fun video of someone using css transforms to simulate 3D physics
    • 1. translate moves an element in the cardinal directions. Done by pixel motion
    • 2. Use webkit-transform for older browsers.
    • 3. Scaling changes the size in both x and y directions.
    • 4. Separate transforms with a list in the same element.
    • 5. Transforms are done in the order you list them. Be careful.
    • 6. Transitions are timed transforms
    • 7. Animations use keyfromes

    • Preface: This is a very varied chapter that has a lot of interesting elements.

    • 2. Height and width need to be set inline rather than in a css file.
    • 3. Canvas uses a coordinate system.
    • 4. SVG is another method of drawing.
