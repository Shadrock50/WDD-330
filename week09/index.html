    Reading 1: Chapter - The Window

    • Preface: Windows pulls away from the DOM.
    • 1. Global Variables use the Window object.
    • 2. Window contains many global functions.
    • 3. Window stores browsing history and can be read: window.history.go(x)
    • 4. Window can create popups
    • 5. Cookies are saved locally and can be read.
    • 6. document.cookie saves or reads the cookie for that page on the localstorage.

    Reading 2: Chapter 14 - API

    • Preface: Everyone loves JSON files.
    • 1. Data attribute embeds information into the page.
    • 2. localstorage.setItem is used to store name/value pairs.
    • 3. set variables directly with localstorage.variable = value.
    • 4. JSON.stringify converts values to json object.
    • 5. JSON.parse converts JSON into variable.
